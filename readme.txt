1. Compilation command
    make clean
    make

2. Execution Step
    java assignmentP1 <InputFileName> 
    Example: java assignmentP1 /home/Username/Documents/neapolitan.txt

3. Output 
   Output will be returned in the second argument given at compile time or if no second argument, than output will be in output.txt file.
   Example: java assignmentP1 input.txt outcome.txt
            java assignmentP1 input.txt
